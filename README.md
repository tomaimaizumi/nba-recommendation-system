# NBA Recommendation System

## 概要
NBA選手のスタッツデータから、類似したプレースタイルの選手を推薦するシステムです。

## 背景
NBAにおける選手比較は主観に依存することが多く、客観的な評価が難しいという課題があります。  
本プロジェクトでは、選手のスタッツデータを用いて特徴量ベクトルを構築し、類似度に基づいた推薦を行うことで、データドリブンな選手分析を実現します。

## システム構成
本システムは以下の流れで構成されています：

1. データ前処理  
   - 欠損値処理  
   - 特徴量の標準化（スケーリング）

2. 特徴量ベクトル化  
   - 各選手を数値ベクトルとして表現

3. 類似度計算  
   - コサイン類似度  
   - ユークリッド距離  

4. 推薦  
   - 入力選手に対して類似度の高い選手を上位K件提示

5. 可視化  
   - Streamlitを用いたUIで結果を表示

## 実行方法

cd nba-recommendation-system

### 2. 必要ライブラリのインストール

pip install -r requirements.txt


### 3. アプリの起動

streamlit run app.py
